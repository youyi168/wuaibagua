# 更新日志

## [v1.0.4] - 2026-03-22

### ✨ 新增
- 📱 **响应式 UI** - 自动适配手机/桌面屏幕
- 🎨 **动态布局** - 支持 320x480 ~ 2560x1440+ 分辨率
- 🔄 **自动调整** - 窗口大小变化时自动优化布局
- 🔍 **网络搜索** - 点击按钮跳转百度搜索卦象详解

### 🐛 修复
- 修复小屏幕设备内容显示不全的问题
- 修复字体大小固定导致的布局问题
- 修复窗口大小变化时 UI 不更新的问题

### 📱 适配设备
- iPhone 13/14/15 (390x844) - 基准尺寸 ✨
- iPhone Pro Max (428x926) - 适度放大
- Android 旗舰 (360-412 x 800+) - 自动适配
- 桌面 (1920x1080+) - 按高度缩放，最大 2 倍

### 🔧 技术改进
- 新增 ResponsiveUI 工具类
- 使用 sp/dp 动态计算字体和组件大小
- 添加窗口大小变化监听
