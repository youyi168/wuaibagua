# 中文字体目录

请将 NotoSansSC-Regular.ttf 或其他中文字体文件放在此目录。

推荐字体：
- NotoSansSC-Regular.ttf (Google Noto Sans SC)
- SourceHanSansSC-Regular.ttf (思源黑体)
- DroidSansFallbackFull.ttf (Android 系统字体)

下载地址：
- Google Fonts: https://github.com/google/fonts/tree/main/apache/notosanscjk
- 清华镜像：https://mirrors.tuna.tsinghua.edu.cn/github-release/google/fonts
