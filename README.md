## 🎯 核心功能

### 起卦功能
- 🎲 **电脑起卦** - 自动投掷三枚铜钱起卦
- ✋ **手动起卦** - 自行选择每次投掷结果
- 📅 **每日运势** - 个性化每日运势（用户识别码系统）
- 🎬 **起卦动画** - 铜钱投掷 + 逐爻显示动画

### 解读功能
- 🔮 **智能解读** - 分 6 类事项精准解读
  - 💰 财运解读
  - 💕 感情解读
  - 💼 事业解读
  - 🏥 健康解读
  - 📚 学业解读
  - ✈️ 出行解读
- 📊 **吉凶判断** - 大吉/吉/平/凶/大凶 + 0-100 分
- 💡 **个性化建议** - 针对性行动建议

### 卦象显示
- 📖 **本卦变卦对比** - 卦名、卦象、卦辞完整对比
- ⚡ **动爻详解** - 每个动爻的本卦 + 变卦 + 解读
- 🎨 **紧凑卦象** - 使用 64 卦 Unicode 符号（䷀ ䷁ ䷂...）
- 📜 **完整爻辞** - 64 卦完整卦辞爻辞，符合《图解周易》

### 辅助功能
- 📝 **占卜事项** - 快捷输入（12 个按钮）+ 手动输入
- 🔍 **网络搜索** - 精准搜索（包含卦名 + 事项）
- 📋 **一键复制** - 复制完整卦象结果
- 📤 **分享功能** - 分享到微信/QQ/剪贴板
- 📜 **历史记录** - 查看、搜索、导出历史记录
- 🌙 **深色模式** - 完全适配的深色/浅色主题

### 收藏与统计
- 📚 **卦象收藏** - 收藏重要卦象，添加标签和备注
- 🏷️ **标签管理** - 按标签搜索和分类
- 📊 **统计图表** - 卦象分布、动爻分布、事项分布
- 📤 **数据导出** - 导出历史记录和收藏为 JSON

### 个性化设置
- 🔊 **音效系统** - 5 种音效，可开关和调节音量
- ⏰ **运势提醒** - 每日定时提醒，防重复
- 🎨 **主题切换** - 深色/浅色模式一键切换

## 📖 使用说明

### 快速起卦

1. **输入事项**（可选）
   - 点击快捷按钮：求财运、问感情、测事业...
   - 或手动输入：例如"求财运"
   
2. **选择起卦方式**
   - 点击「📅 今日运势」- 获取今日个性化运势
   - 点击「电脑起卦」- 随机起卦（带动画和音效）
   - 点击「手动起卦」- 逐爻投掷

3. **查看解读**
   - 吉凶判断：大吉 ✅ (85 分)
   - 卦象分析：本卦、变卦、动爻
   - 事项解读：针对性解读
   - 完整爻辞：卦辞爻辞详情

4. **操作**
   - 🔍 搜索 - 精准搜索卦象详解
   - 📋 复制 - 复制完整结果（带音效）
   - 📤 分享 - 分享到微信/QQ
   - 📚 收藏 - 收藏卦象，添加标签
   - 📜 历史 - 查看历史记录
   - 📊 统计 - 查看统计图表

### 断卦规则

遵循《图解周易》传统规则：

| 动爻数 | 断卦规则 |
|--------|---------|
| 0 爻动 | 六爻皆静，以本卦卦辞断之 |
| 1 爻动 | 一爻动，以动爻爻辞断之 |
| 2 爻动 | 两爻动，一阴一阳以阴爻为主，同阴/阳以上爻为主 |
| 3 爻动 | 三爻动，取中间爻断之 |
| 4 爻动 | 四爻动，看下静爻断之 |
| 5 爻动 | 五爻动，看静爻断之 |
| 6 爻动 | 六爻皆动，乾用九/坤用六/他卦看变卦 |

## 🎯 使用场景

### 场景 1: 求财运
```
输入：求财运
起卦：天火同人
吉凶：大吉 ✅ (85 分)
解读：财运较好，适合投资理财，但需理性分析...
操作：收藏 → 添加标签"财运"、"重要"
```

### 场景 2: 问感情
```
输入：问感情
起卦：泽山咸
吉凶：吉 👍 (70 分)
解读：感情运势良好，单身者有机会遇到合适对象...
操作：分享 → 微信好友
```

### 场景 3: 每日运势
```
点击：📅 今日运势
结果：同一用户今日结果相同
可验证：多次查看结果一致
提醒：每日 08:00 自动提醒
```

### 场景 4: 查看统计
```
点击：📊 统计
查看：
- 卦象分布（乾为天 8 次最多）
- 动爻分布（2 爻动最常见）
- 事项分布（求财运 5 次最多）
```
