# 我爱八卦 APK 编译指南

## 环境要求

### 已安装组件
- ✅ Android SDK (~/Android/Sdk)
- ✅ Android NDK r25b (~/Android/ndk)
- ✅ Buildozer 1.5.0
- ✅ python-for-android 2024.1.21
- ✅ Java JDK 17

## 本地编译步骤

### 方法 1：使用 Buildozer（推荐）

```bash
cd /home/admin/openclaw/workspace/temp/wuaibagua

# 1. 清理旧的构建缓存
rm -rf .buildozer

# 2. 设置环境变量
export ANDROID_SDK_ROOT=~/Android/Sdk
export ANDROID_NDK=~/Android/ndk
export NDK_HOME=~/Android/ndk

# 3. 开始编译（调试版）
buildozer android debug

# 或者编译发布版（需要签名）
buildozer android release
```

### 方法 2：直接使用 p4a

```bash
cd /home/admin/openclaw/workspace/temp/wuaibagua

python3 -m pythonforandroid.toolchain apk \
    --dist_name=woaibagua \
    --bootstrap=sdl2 \
    --requirements=python3,kivy,pyjnius==1.6.1 \
    --arch=arm64-v8a \
    --arch=armeabi-v7a \
    --android_api=29 \
    --ndk_api=21 \
    --private=/home/admin/openclaw/workspace/temp/wuaibagua \
    --package=org.woaibagua.app \
    --name="我爱八卦" \
    --version=1.0.0 \
    --permission=INTERNET
```

## 编译输出

编译成功后，APK 文件位置：
- **调试版**: `bin/woaibagua-1.0.0-debug.apk`
- **发布版**: `bin/woaibagua-1.0.0-release.apk`

## 常见问题解决

### 问题 1：下载失败（权限问题）

```bash
# 确保构建目录有写权限
chmod -R 777 .buildozer
mkdir -p .buildozer/android/platform/build-arm64-v8a_armeabi-v7a/packages
```

### 问题 2：NDK 未找到

```bash
# 检查 NDK 路径
ls -la ~/Android/ndk

# 如果不存在，重新下载
cd /tmp
wget https://dl.google.com/android/repository/android-ndk-r25b-linux.zip
unzip android-ndk-r25b-linux.zip
mv android-ndk-r25b ~/Android/ndk
```

### 问题 3：Python 版本不兼容

确保使用 Python 3.10：
```bash
python3 --version
# 应该是 Python 3.10.x
```

### 问题 4：Cython 版本问题

```bash
pip3 install --upgrade cython
```

## 配置文件说明

### buildozer.spec 关键配置

```ini
[app]
title = 我爱八卦
package.name = woaibagua
package.domain = org.woaibagua
version = 1.0.0
requirements = python3,kivy,pyjnius==1.6.1
orientation = portrait
android.api = 29
android.ndk_api = 21
android.archs = arm64-v8a, armeabi-v7a
```

## 编译时间预估

首次编译（需要下载所有依赖）：约 30-60 分钟
后续编译（使用缓存）：约 10-20 分钟

```bash

adb devices

# 安装 APK
adb install bin/woaibagua-1.0.0-debug.apk
```

## 注意事项

1. **网络要求**：首次编译需要下载大量依赖，确保网络畅通
2. **磁盘空间**：至少需要 5GB 可用空间
3. **内存要求**：建议至少 4GB RAM
4. **权限问题**：确保对构建目录有完全访问权限
