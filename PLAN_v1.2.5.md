def on_copy(self, instance):
    """复制卦象释义"""
    if not self.current_result:

    self.share_manager.copy_to_clipboard(text)
    self.show_toast('✅ 已复制到剪贴板')

## 🔧 代码修改计划

### 1. 修改 MainScreen 布局

**文件**: `wuaibagua_kivy.py`

**修改内容**:
```python
# 调整各区域 size_hint_y
gua_layout = BoxLayout(size_hint_y=0.30)  # 38% → 30%

```

### 2. 优化释义显示组件

**新增**: `GuaTextDisplay` 组件

**功能**:
- 支持 Markdown 格式
- 自动换行
- 分段显示
- 支持复制

### 3. 添加复制按钮

**位置**: 工具栏（主题切换 + 分享 + 复制）

**代码**:
```python
self.btn_copy = Button(
    text="📋 复制",
    font_size=self.responsive.get_font_size(15),
    background_color=(0.4, 0.4, 0.4, 1)
)
self.btn_copy.bind(on_press=self.on_copy)
```

### 4. 优化数据加载

**改进**:
```python
# 当前：一次性加载所有卦象数据
# 优化：按需加载，提升启动速度

def load_gua_on_demand(self, gua_name):
    """按需加载卦象数据"""
    if gua_name not in self.gua_data:
        self.gua_data[gua_name] = self.cache_manager.get(...)
    return self.gua_data[gua_name]
```
