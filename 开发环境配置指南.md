## 📝 配置文件说明

### buildozer.spec

关键配置项：

```ini
[app]
title = 吾爱八卦                  # 应用名称
package.name = wuaibagua          # 包名
package.domain = org.wuaibagua    # 域名
source.dir = .                    # 源代码目录
version = 2.4.0                   # 版本号
requirements = python3,kivy       # 依赖
orientation = portrait            # 屏幕方向

android.api = 29                  # Android API 级别
android.minapi = 21               # 最低支持 API
android.ndk_api = 21              # NDK API 级别
android.ndk = 25b                 # NDK 版本
```

## ⚠️ 常见问题

### 1. 编译失败 - 网络问题

**症状**：下载 Android SDK/NDK 失败

**解决**：使用国内镜像

```bash
# 编辑 buildozer.spec
nano buildozer.spec

# 添加镜像配置
p4a.source_url = https://gitee.com/mirrors/python-for-android.git
```

### 2. 汉字显示乱码

**症状**：APK 中汉字显示为方框或乱码

**解决**：已配置中文字体

- 确保 `fonts/NotoSansSC-Regular.ttf` 存在
- `buildozer.spec` 中配置 `android.add_assets = fonts`
- `main.py` 中已注册中文字体

### 3. 磁盘空间不足

**症状**：编译过程中提示空间不足

**解决**：

```bash
# 清理构建缓存
buildozer clean

# 检查磁盘使用
df -h

# 建议保留至少 10GB 可用空间
```

### 4. Java 版本错误

**症状**：提示 Java 版本不兼容

**解决**：

```bash
# 检查 Java 版本
java -version

# 应该是 OpenJDK 17
# 如果不是，重新加载环境变量
source ~/.bashrc
```

配置文件：`~/.pip/pip.conf`

```ini
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/
trusted-host = mirrors.aliyun.com
```

### Android SDK 镜像

Buildozer 会自动下载 Android SDK，使用以下镜像：

- SDK: 自动从 Google 下载（首次约 1-2GB）
- NDK: r25b（约 800MB）
